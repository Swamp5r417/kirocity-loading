KIROCITY+ Loading Screen v14

Что изменено:
- Когда загрузка доходит до финальной стадии, весь интерфейс и фон исчезают.
- После этого появляется полностью чёрный экран.
- На чёрном экране выводится надпись "WELCOME TO".
- Ниже загорается "KIROCITY" с эффектом старой лампы / старого света.
- Если Garry's Mod не передаст финальный статус, есть запасной таймер, который тоже запускает финальную сцену.

Как поставить:
1. Распакуй ZIP.
2. Загрузи всю папку на хостинг.
3. В server.cfg поставь:
   sv_loadingurl "https://твой-сайт/index.html"

Если хочешь изменить момент появления финальной сцены:
- открой script.js
- измени значение finalFallbackDelay
